### Luo's Spider Project v0.0.1

#### release date
 2021年1月21日


#### introduction
这个项目用于收集Jarrett完成的各类爬虫

- douban爬虫
    用于爬各种电影评论，评分等。
- FindJobfromSichuan
    用于爬四川事业单位招聘网的各种招聘信息，并保存为不同类型的文件。
    
- lengxiaohua_spyder
    冷笑话网的爬虫，爬取热门的冷笑话。并可以整理冷笑话中出现的高频词。
    
- QingyunianfromWeibo
    庆余年的微博评论爬取。
